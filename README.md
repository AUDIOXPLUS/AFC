# AFC - Audio File Converter

## Descrizione
AFC è un'applicazione web per la gestione e la conversione di file audio. Permette agli utenti di caricare, organizzare e gestire progetti audio attraverso un'interfaccia intuitiva.

## Funzionalità Principali
- Gestione progetti audio
- Sistema di autenticazione utenti
- Organizzazione in fasi di lavoro
- Gestione ruoli e permessi
- Upload e gestione file
- Interfaccia responsive

## Tecnologie Utilizzate
- Node.js
- Express.js
- SQLite
- HTML/CSS/JavaScript

## Installazione
```bash
# Installa le dipendenze
npm install

# Avvia il server
npm start
```

## Struttura del Progetto
- `/database`: Contiene il database SQLite
- `/uploads`: Directory per i file caricati
- File statici (HTML, CSS, JS) per l'interfaccia utente
